# Project Pwner

Monorepo scaffold for Project Pwner MVP.

## Workspace layout
- `apps/web` – placeholder Next.js app target
- `apps/api` – Fastify API skeleton + DB migrations
- `packages/shared` – shared types/contracts

## Commands
- `pnpm dev` – run API dev server
- `pnpm test` – run API tests
- `pnpm db:migrate` – apply DB migrations for API
